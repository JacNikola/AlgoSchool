What is an Algorithm?

An algorithm is a set of well-defined steps, written to solve a problem or to accomplish a particular task. The recipe of a dish is a good example of an algorithm.

What is Programming?

Programming (also called Coding) is basically the process of encoding the human readable algorithm into a notation, a programming language (such as C++, Java, Python, etc.), so that it can be executed by a computer.

What makes it exciting?

The joy of solving seemingly difficult problems, completely using logic and intuition is what makes programming an exciting field of study. Unlike other physical sciences, where the solution to a problem might be restricted by the laws of Physics, in Programming, the only barrier to coming up with a solution will be your thinking ability. Thus the prerequisite for getting started with programming is very low, and literally anyone, with a knack for creativity and logical thinking, can start enjoying the essence of Algorithms and Programming.
